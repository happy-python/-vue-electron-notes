# vue-electron-notes

## 安装依赖
```
yarn install
```

### 运行
```
yarn electron:serve
```

### 打包
```
yarn electron:build
```

### 自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).

### 第三方库
    - UI：element-ui
    - 本地存储：nedb，nedb-promises
    - 图标：fontawesome
    - markdown：mavon-editor
